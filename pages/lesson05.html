Заняття 5: Цикли в Python

1. Вступ

Цикли дозволяють повторювати блок коду кілька разів. У Python є два основні типи циклів:

  • for – для ітерацій по елементах або числовому діапазону;
  • while – виконується, поки умова є істинною.
