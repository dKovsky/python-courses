Заняття 1: Вступ до Python та основи синтаксису

Ввід і вивід даних та базовий синтаксис

Значення, змінні, та іменування

Значення — це конкретні дані, які зберігаються в програмі. Значення можуть бути числами, текстом, логічними величинами або іншими типами даних. Наприклад:

  • Ціле число: 42
  • Число з плаваючою крапкою: 3.14
  • Рядок: "Hello"
  • Булеве значення: True або False

Змінна — це іменована комірка пам’яті для збереження даних. Правила іменування:

  • Можна використовувати букви, цифри, _, але не починати з цифри
  • Чутливе до регістру: age ≠ Age
  • Не використовувати ключові слова Python (if, for, def)
  • Краще зрозумілі імена: name, age, total_score

Функція print()

print() використовується для виведення інформації на екран.

Функція input()

input() отримує дані від користувача. Завжди повертає рядок (str).

Числові типи даних

1. Задання чисел

  1. 1.1 int (цілі числа)
  2. 1.2 float (числа з плаваючою крапкою)
  3. 1.3 complex (комплексні числа)

2. Операції з числовими типами:

  • Додавання: x + y
    Результат: 100 + 3 = 103
  • Віднімання: x - y
    Результат: 100 - 3 = 97
  • Множення: x * y
    Результат: 100 * 3 = 300
  • Ділення: x / y (результат завжди типу float)
    Результат: 100 / 3 = 33.333333333333336
    Нюанс: Ділення завжди повертає результат типу float, навіть якщо обидва операнди — цілі числа.
  • Ціле ділення: x // y (результат типу int, якщо обидва операнди — цілі числа)
    Результат: 100 // 3 = 33
    Нюанс: Якщо хоча б один з операндів є float, результат буде типу float: 
    
x = 10.0
y = 2
result = x // y  # Результат: 5.0

    Однак, якщо обидва операнди цілих чисел, результат буде типу int.
    Нюанс з негативними числами: Якщо один з операндів є від'ємним, результат ціле ділення буде округлений у бік мінус нескінченності. Тобто результат буде меншим або рівним дійсному результату звичайного ділення: 
    
x = -10
y = 3
result = x // y  # Результат: -4

    Це означає, що результат округляється до найближчого більшого цілого числа в бік мінус нескінченності. В даному прикладі результат ділення `-10 / 3` є приблизно `-3.3333...`, але результат ціле ділення — це `-4`.
  • Модуль: x % y
    Результат: 100 % 3 = 1
  • Піднесення до степеня: x ** y
    Результат: 100 ** 3 = 1000000

1. str (рядки)

Рядки в Python — це послідовність символів, які можна оголошувати в одинарних ' ', подвійних " ".

Для багаторядкових рядків потрійних одинарних ''' ''' або подвійних """ """ лапках.

  • Перехід на новий рядок: \n
  • Табуляція: \t

  • Гарячі лапки (дві лапки): \"
  • Одинарні лапки: \'
  • Юнікод-символи: \u або \U

5. f-string (форматування рядків у Python 3.6+)

Дозволяє вставляти змінні та вирази безпосередньо в рядок.

      
name = "world"
year = 2025
text = f"Hello, {name}! Year: {year}"
print(text)  # 'Hello, world! Year: 2025'

6. r-string (сирий рядок)

Використовується для обробки спеціальних символів (наприклад, шляхів файлів), не обробляючи зворотні слеші.

      
path = r"C:\Users\Python\Documents"
print(path)        # 'C:\Users\Python\Documents'
print(repr(path))  # 'C:\\Users\\Python\\Documents'

SQL = r'''
SELECT *
FROM tbl
WHERE col = 'value'
'''
print(SQL)
print(repr(SQL))
Виведення SQL-запиту:
      
# Виведення SQL-запиту, як він є, з новими рядками

Логічний тип даних

Булівські типи в Python використовуються для зберігання значень, які можуть бути тільки двох можливих варіантів: True або False. Вони є основою для логічних операцій та умовних конструкцій.

Булівські значення можна створювати за допомогою ключових слів True і False.
