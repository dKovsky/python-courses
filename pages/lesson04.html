Заняття 4: Булевий тип та if

Булівські типи в Python використовуються для зберігання значень, які можуть бути тільки двох можливих варіантів: True або False. Вони є основою для логічних операцій та умовних конструкцій.

1. Створення булівських значень

Булівські значення можна створювати за допомогою ключових слів True і False.

2. Операції з булівськими значеннями

Булівські значення зазвичай використовуються для порівняння та логічних операцій.

3. Перетворення в булевий тип

В Python можна перетворювати різні типи в булеві значення за допомогою функції bool().

Значення, які дають False: None, 0, порожні контейнери (наприклад, порожні списки [], порожні рядки "", порожні множини set()).

Інші значення дають True.

6. Підсумок

  • Булівські значення можуть бути лише True або False.
