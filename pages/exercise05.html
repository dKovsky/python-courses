Завдання № 6.

Скрипт має отримувати команди від користувача,
якщо користувач вкаже команду exit програма має завершитись;
для команди hi - вивести 'Привіт'.
Приклад:
app -> Введіть команду:
user -> new
app -> Введіть команду:
user -> hi
app -> Привіт
app -> Введіть команду:
user -> 1 + 1
app -> Введіть команду:
user -> exit
app -> Програму завершено
