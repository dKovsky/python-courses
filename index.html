Python

Програма занять

У курсі буде

  • Основи Python: Вивчення базового синтаксису мови, робота з основними типами даних: числа, рядки, списки, словники, множини та кортежі.
  • Робота з файлами: Читання, запис і обробка текстових (TXT), табличних (CSV) та структурованих (JSON) файлів.
  • Логіка програм: Умовні оператори (if, else, elif) та цикли (for, while).
  • Обробка помилок: Використання try / except для запобігання аварійному завершенню програм.
  • Робота з файлами та папками: Модуль os — створення, видалення, копіювання та перевірка файлів і папок.
  • Робота з базами даних: Підключення до Oracle, PostgreSQL та SQL Server, виконання SQL-запитів.

Популярні бібліотеки Python

  • openpyxl — робота з Excel-файлами (.xlsx), зчитування, запис і редагування табличних даних.
  • pandas — аналіз та обробка великих обсягів даних (CSV, Excel, JSON, бази даних).
  • paramiko — робота з SFTP-серверами, передача файлів через захищене з'єднання.
  • pywin32 — автоматизація Windows-задач, зокрема відправка електронних листів через Outlook.

Практика: створення програм для відправки email через Outlook та роботи з SFTP-серверами.
